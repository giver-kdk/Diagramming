- 'include' means compulsary task that needs to be done to complete the main task
	- For eg: We need to login to purchase product from website
	- NOTE: The 'include' should be used with dashed line

- 'extend' means the task that needs to be done on failure/alternative(Like error handling)
	- For eg: Redirecting customer to forget password page if they forget it
	- NOTE: The 'extend' should also be used with dashed line
